Instructions

The interface: On the top a search bar that lets you look for a specific institution. By clicking "add it" the Dynamic network (right side, or second graph below in mobile) will update only displaying the selected Institution and its adiacent nodes. At the same time, the bar chart on the left (or the first graph in mobile) will highlight the techniques taught by that institution using the same color of the institution. At the moment you can add up to 2 institutions with the Add it button. If two institutions share the same technique then a "both" color will appear as well and on the left graph the shared techniques will be highlighted with that color.

On the left (or first graph in mobile): A bar chart that shows for every tadirah technique (from the original DH Registry Dataset) The number of articles written in DH Journals. By clicking on that technique, a new window will pop up with all the articles related to that technique. When selecting an institution, the techniques taught by its courses will be highlighted. The plotly library allows the user to select portion of the graph, in order to return to the normal state a double click is needed.

On the Right (or second graph below in mobile): A dynamic network of publications of the institutions contained in the original dataset. The default one contains all the institutions, but some subnetworks can be extracted by selecting institutions with the search bar above. Clicking on an institution node opens a tab with all the detected publications of that institution along with its DH Courses. Clicking on an arch between two institutions will open a tab that shows the publications co-authored by people affiliated to both institutions.
